HTML Basic Tag

HTML (Hypertext Markup Language)

HTML (Hypertext Markup Language): Bahasa markup yang digunakan untuk membuat halaman web dan menampilkan informasi di browser. HTML terdiri dari tag yang memberi tahu browser cara menampilkan konten. Standar HTML dikelola oleh W3C.

Sejarah HTML dan XHTML

Dimulai dengan HTML 2. 0 yang distandarisasi pada tahun 1995 (dan yang sekarang kita kenal dan digunakan saat saat ini, HTML 5.0), kemudian dikembangkan lebih lanjut pada tahun 2007 dengan XHTML 1. 1 yang menggabungkan HTML dan XML.

Struktur HTML

HTML memiliki tiga bagian utama: <html></html>, <head></head>, dan <body></body>. <html> berfungsi sebagai elemen root atau pembungkus utama dari seluruh dokumen HTML, <head> berisi informasi tentang dokumen, dan <body> menampilkan konten untuk pengguna.

Tag HTML

  • Heading: Menggunakan tag <h1></h1> hingga <h6></h6> untuk membuat judul dan subjudul.
  • Paragraf: Tag <p></p> digunakan untuk menulis paragraf. <br /> untuk line break, dan <hr /> untuk garis horizontal.
  • Font: Tag seperti <b></b>, <i></i>, dan <u></u> digunakan untuk menebalkan, memiringkan, dan menggarisbawahi teks.
  • Image: Tag <img> digunakan untuk menampilkan gambar dengan atribut seperti src, width, height, alt, dan title.
  • Anchor/Hyperlink: Tag <a></a> digunakan untuk membuat tautan ke halaman lain atau bagian dari halaman yang sama.
  • Komentar: Menggunakan <!-- komentar --> untuk menyisipkan catatan yang tidak ditampilkan oleh browser.
